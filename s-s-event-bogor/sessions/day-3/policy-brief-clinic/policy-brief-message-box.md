## Quick check

Before you move on, test your Message Box:

- Can someone outside your field understand the So What in one reading?
- Does your Issue fit in a single short phrase?
- Are your Solutions specific to what ocean accounts can show (not general wishes)?
- Would your target audience recognize themselves in the Problems section?

# Worked example: Ocean Accounts and the Global Biodiversity Framework

**Target audience:** CBD national focal point / Ministry of Environment NBSAP team

```
+-------------------------------------------------------------------+
|                                                                   |
|                         PROBLEMS                                  |
|                                                                   |
|  Countries lack standardized data to report on GBF targets.       |
|  Biodiversity data is fragmented across agencies and sectors.     |
|  Current monitoring systems do not connect environmental          |
|  trends to economic or social outcomes.                           |
|                                                                   |
+-------------------+                       +-----------------------+
|                   |                       |                       |
|    BENEFITS       |       ISSUE           |     SO WHAT           |
|                   |                       |                       |
|  Countries can    |  Using ocean          |  Without consistent   |
|  monitor NBSAPs   |  accounts to track    |  measurement,         |
|  with standardized|  progress on the      |  countries cannot     |
|  data. Spatial    |  Global Biodiversity  |  track whether the    |
|  planning becomes |  Framework            |  30x30 target is      |
|  evidence-based.  |                       |  being met, whether   |
|  Restoration      |                       |  conservation         |
|  priorities are   |                       |  spending is          |
|  identified.      |                       |  effective, or        |
|  Reporting to the |                       |  whether marine       |
|  CBD becomes      |                       |  ecosystems are       |
|  comparable.      |                       |  improving.           |
|                   |                       |                       |
+-------------------+                       +-----------------------+
|                                                                   |
|                        SOLUTIONS                                  |
|                                                                   |
|  Ocean accounts integrate environmental, economic, and social     |
|  data under the SEEA framework. Extent and condition accounts     |
|  track habitat area and ecosystem health. Services accounts       |
|  measure benefits to people. All are aligned with GBF targets     |
|  1 through 14 and can feed directly into NBSAP reporting.         |
|                                                                   |
+-------------------------------------------------------------------+
```

Notice what is not in this box: no methods, no technical acronyms beyond what the audience knows (GBF, NBSAP, SEEA), no literature review. The entire message fits on one page.

Source: GOAP (2024). Using Ocean Accounts to Monitor Progress Towards Global Biodiversity Framework (GBF) Goals. GOAP Subject Brief.
