Economic Data

Prices, costs, revenues

NCP rates, value transfer

Regulating

  • Nursery habitat support
  • Water purification

Cultural

  • Tourism and recreation
  • Gleaning (subsistence)
  • Cultural and spiritual
Valuation Methods

Resource Rent

Revenue minus costs

Social Cost of Carbon

Replacement Cost

Cost of built alternative

Direct Expenditure

Actual spending data

Value Transfer

Apply rates from similar sites

Outputs

Physical Supply Table

Service x ecosystem type x quantity (kg, Mg CO2, visits, km)

Monetary Supply Table

Service x ecosystem type x value ($/yr)
