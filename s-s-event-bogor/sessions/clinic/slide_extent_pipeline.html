Extent Account Pipeline

Two pathways from data to SEEA EA Extent Account
A No own data Tier 1
1
Global datasets
Allen Coral Atlas, Global Mangrove Watch, Global Fishing Watch, etc.
2
Define accounting area
Set national / subnational boundary for the study area
3
Clip and extract
Extract extent per ecosystem type within the boundary
4
Fill extent account table
Opening/closing balances from available time points
B Own data available Tier 2
1
Satellite imagery + ground truth
Sentinel-2 / Planet + in-situ validation points
2
Preprocessing
Atmospheric correction, sun-glint removal, water column correction
3
Classification + accuracy
SAM / Random Forest classifier + accuracy assessment (confusion matrix)
4
Extent account + change matrix
Full opening/closing table with cross-tabulated change detection

SEEA EA Extent Account

Standardised table: ecosystem type x area (ha) x opening/closing balance x additions/reductions
