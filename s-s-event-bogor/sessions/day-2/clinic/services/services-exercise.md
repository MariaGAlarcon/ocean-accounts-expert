# INTERPRETATION

1. Which service has the highest physical supply (in its own units)? Is this the same one with the highest monetary value? Why might they differ?

2. Carbon sequestration uses SCC (USD 51/Mg CO2). Recalculate using a voluntary market price of USD 15/Mg CO2. How does the value change? Which price is more appropriate for an NDC report vs. a carbon credit project?

3. Could you report Part A without Part B? In what policy context would the physical supply table alone be useful?

4. The coastal protection value uses replacement cost. Does a seawall replace all the services a coral reef provides? What services would be lost even if a seawall were built?

5. If fishing costs rose to USD 600,000, what happens to the resource rent? Does this mean the ecosystem stops providing fish?

## Answer Key

### Part A

| Service | Physical supply |
|---------|---:|
| Carbon (mangrove) | 350 x 3.7 = **1,295 Mg CO2/yr** |
| Carbon (seagrass) | 800 x 1.3 = **1,040 Mg CO2/yr** |
| Carbon total | **2,335 Mg CO2/yr** |
| Fish provisioning | **180,000 kg/yr** |
| Coastal protection | **12,000 m** |
| Recreation | **15,000 visitors/yr** |

### Part B

| Service | Value (USD/yr) |
|---------|---:|
| Fish: revenue = 180,000 x 3.50 | 630,000 |
| Fish: resource rent = 630,000 - 420,000 | **210,000** |
| Carbon: mangrove = 1,295 x 51 | **66,045** |
| Carbon: seagrass = 1,040 x 51 | **53,040** |
| Carbon total | **119,085** |
| Recreation = 15,000 x 85 | **1,275,000** |
| Coastal: total = 12,000 x 5,000 | 60,000,000 |
| Coastal: annualized = 60,000,000 / 50 | **1,200,000** |
| **TOTAL** | **2,804,085** |

### Interpretation answers

1. Carbon has the largest physical supply in absolute terms (2,335 Mg CO2/yr) but the lowest monetary value (USD 119K). Recreation has the highest monetary value (USD 1.28M) from only 15,000 visitors. Physical and monetary rankings differ because they depend on the price/method used, not just the quantity.

2. At USD 15: 2,335 x 15 = USD 35,025/yr (vs. USD 119,085 at SCC). SCC is appropriate for NDC reporting (reflects global climate damages). Market price is appropriate for a carbon credit project (reflects what buyers will pay).

3. Yes. Physical supply alone is useful for SDG 14 reporting (tonnes of fish, hectares of ecosystem), MPA management plans (visitor carrying capacity), and climate reporting (Mg CO2 sequestered for NDC).

4. No. A seawall provides wave attenuation only. It does not provide fish habitat, nursery function, carbon storage, biodiversity, tourism value, or sediment production. Replacement cost is a lower bound on one function of the reef.

5. Resource rent = 630,000 - 600,000 = USD 30,000. If costs exceeded 630,000, resource rent = 0 (capped). The ecosystem still provides fish (the physical supply is unchanged). Zero resource rent means the commercial fishery is unprofitable, not that the fish have no value. Subsistence and food security value is not captured by resource rent.
