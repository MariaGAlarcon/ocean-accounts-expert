# Ecosystem Services Accounts -- Clinic Walkthrough Slides

**Time:** 15-30 min depending on participant level
**Purpose:** Guide a fellow through building an ecosystem services account

---

## Slide 1: What is a services account?

An ecosystem services account quantifies the **benefits that ecosystems provide to people**. It has two distinct parts that can be done independently:

| Part | What it measures | Unit examples | Can stand alone? |
|------|-----------------|--------------|-----------------|
| **Part A: Physical quantification** | How much service the ecosystem provides | kg fish/yr, Mg CO2/yr, visitors/yr, m coastline | Yes |
| **Part B: Monetary valuation** | Economic significance of that service | USD/yr (exchange values) | Requires Part A first |

**Part A is a complete account on its own.** Many countries start with physical quantification and add monetary valuation later when economic data become available.

## Slide 3: Part A -- Physical quantification

**This is the foundation. It answers: how much service does the ecosystem provide?**

Three pathways depending on your data:

| Data situation | Approach | Example |
|---------------|----------|---------|
| No primary data | Literature rates x extent | Carbon: 3.7 Mg CO2/ha/yr x 350 ha mangrove |
| Government statistics | Use existing records | Fisheries: national catch data by species |
| Primary field data | Measure directly | Tourism: visitor counts from dive centres |

**Physical supply table (your output):**

| Service | Unit | Coral reefs | Seagrass | Mangroves | Total |
|---------|------|---:|---:|---:|---:|
| Fish provisioning | kg/yr | | | | |
| Carbon sequestration | Mg CO2/yr | | | | |
| Coastal protection | m coastline | | | | |
| Recreation | visitors/yr | | | | |

**This table is a valid account. You can stop here if you do not have economic data.**

## Slide 4: Part B -- Monetary valuation (optional, builds on Part A)

**Only after Part A is complete.** Assigns economic values to the physical quantities.

| Method | When to use | Service examples | Value type |
|--------|------------|-----------------|-----------|
| **Resource rent** | Marketed goods | Fish, timber | Market-based |
| **Social cost of carbon** | Climate regulation | Carbon sequestration | Non-market |
| **Replacement cost** | Protective services | Coastal protection, sediment | Non-market |
| **Direct expenditure** | Tourism/recreation | Reef diving, mangrove tours | Market-based |
| **Equivalent wage** | Unpaid activities | Gleaning, subsistence fishing | Non-market |
| **Value transfer** | No local data | Any service (Tier 1 estimate) | Transferred |

**Important:** Not all values are market prices. Replacement cost and SCC are estimated values, not observed transactions. The SEEA EA requires exchange values (what a buyer and seller would agree), not welfare values (total willingness to pay).

## Slide 5: Logic chains -- how Part A flows into Part B

Each service follows a logic chain: **data in, physical quantity out, then value.**

**Carbon sequestration:**
```
Extent (ha) --> NCP rate (Mg CO2/ha/yr) --> Physical supply (Mg CO2/yr) --> x SCC --> Value (USD/yr)
     350              3.7                        1,295                    x $51     $66,045
```

**Fish provisioning:**
```
Catch data (kg) --> x Price (USD/kg) --> Revenue --> minus Costs --> Resource rent (USD/yr)
   180,000             x $3.50          $630,000      $420,000         $210,000
```

**Coastal protection:**
```
Reef extent along coast (m) --> Assets behind reef --> Replacement cost (USD/m) --> Annualized
        12,000 m                    buildings               $5,000              $1,200,000/yr
```

**Recreation:**
```
Visitors/yr --> x Reef-attributable spending --> Value (USD/yr)
  15,000                   $85                    $1,275,000
```

## Slide 6: What to do right now

**Step 1: Choose your services (based on policy question)**

| Policy priority | Start with |
|----------------|-----------|
| Climate / NDC | Carbon sequestration (Part A is straightforward) |
| Food security | Fish provisioning |
| Coastal resilience | Coastal protection |
| Tourism / blue economy | Recreation |
| MPA justification | Multiple services |

**Step 2: Do Part A (physical quantification)**
- Use the guide for your chosen service
- Fill in the physical supply table
- This is a valid account

**Step 3: Decide if you can do Part B (monetary)**
- Do you have price data, cost data, or valuation studies?
- If yes, apply the appropriate method
- If no, report Part A only. It is still useful for policy.

**Step 4: Apply to your data**
- Use the exercise to practice with a worked scenario
- Then open the Excel template and enter your own numbers
