# Print Notebooks -- Instructions for Printing

## Overview

7 notebooks, one per theme. Print each notebook as a separate bound booklet.
Files are numbered in the order they should be printed/bound.

**Total: approximately 152 pages across all notebooks.**

## File types

| Extension | How to print |
|-----------|-------------|
| .docx | Print directly from Word. A4, portrait. |
| .pdf | Print directly. These are visual slides, print landscape or as-is. |
| .png | Print as images. Scale to fit A4 page, landscape orientation. |
| .xlsx | Do NOT print. These are digital companion files (templates and exercise workbooks for laptop use). |

## Binding suggestions

- Print each notebook separately
- Color printing for .pdf and .png files (the visuals use GOAP color coding)
- Black and white is fine for .docx files (tables have structural formatting)
- Spiral binding or stapled booklets
- Consider printing the 00-overview notebook on slightly thicker paper or with a colored cover since it is the main reference
