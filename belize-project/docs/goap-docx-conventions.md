Reference for generating .docx files that match the GOAP Report Template. Table formatting rules are in CLAUDE.md (the authoritative source). This file covers colors, typography, and page layout.

## Page Layout (A4)

- Page size: A4 (210 x 297 mm)
- Portrait margins: top 1.19 in, bottom/left/right 0.91 in
- Landscape margins: top/bottom/right 0.91 in, left 1.19 in
- Headers and footers: 8 pt Arial, #30302F

## Table Style Variants

The GOAP template includes several table color schemes. The default (green theme) is defined in CLAUDE.md. Additional variants:

**Light green theme**: header fill #3B9C7B (first col), #D4EEE5 (data cols).

**Blue theme**: header fill #66A3B5 (first col), #A3C7D2 (data cols).

**Multi-header theme**: category header #0A5455, sub-header #3B9C7B.
